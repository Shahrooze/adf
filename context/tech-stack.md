# Design

Design Tokens map onto

- Tailwind CSS theme configuration
- shadcn/ui component primitives

Design Tokens cover

- Color roles
- Typography scale
- Spacing scale
- Radius scale
- Elevation / shadow scale

The Design Agent must express tokens in terms that translate directly into
this configuration. The Frontend Agent must not introduce values outside the
token scale.

# AI Rules

Agents MUST

- Reuse existing libraries.
- Avoid unnecessary dependencies.
- Prefer framework defaults.
- Avoid experimental packages.
- Prefer LTS technologies.
- Generate production-ready code.

Agents MUST NOT

- Introduce new frameworks without justification.
- Mix architectural styles.
- Generate obsolete code.
- Use deprecated APIs.
